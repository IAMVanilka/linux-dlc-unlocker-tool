# Changelog

## [1.1.0] - 2026-04-20
### Added
- Проверка хэшэй файлов DLC. Теперь утилита скачивает только тот контент, которого недостаёт.
- Флаг `--force` позволяет скачать и распаковать DLC даже, если они уже имеются в папке с DLC. 


## [1.0.0] - 2026-04-18
### Changed
- Общий рефакторинг кода.
- Изменен метод установки утилиты (*теперь всё делается через `pipx/pip`*).
- Код покрыт тестами (*С помощью ИИ*).
- Изменен способ загрузки **DLC**. Теперь всё делается силами python без **wget** как было раньше.
- Начался отсчёт версий.
