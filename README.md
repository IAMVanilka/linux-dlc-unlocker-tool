# Linux DLC Unlocker Tool (ldu-tool)

Утилита на Python для автоматической разблокировки всех DLC игры **Stellaris** в Linux-среде (Steam). Скрипт использует эмулятор Steam API (Goldberg Emulator) и автоматически загружает необходимые файлы контента.

> ⚠️ Данная утилита не предназначена для анлока DLC на **windows**! Аналогичный софт для windows можно найти в **[данном репозитории](https://github.com/seuyh/stellaris-dlc-unlocker)**.
> 
> Для скачивания DLC данная утилита использует удаленный файловый сервер из **[того же репозитория](https://github.com/seuyh/stellaris-dlc-unlocker)**. Спасибо [seuyh](https://github.com/seuyh) за предоставленные удобства.

## 📋 Требования

- **Python 3.6+**

- **Библиотека `requests`**

- **`wget`** (стандартная утилита Linux для загрузки файлов по HTTP/HTTPS)

- Доступ к интернету (для доступа к `api.steamcmd.net` из некоторых регионов, включая РФ, может потребоваться VPN)

- Права на запись в директорию с установленной игрой

## 🚀 Установка и запуск
1. [Скачайте утилиту](https://raw.githubusercontent.com/IAMVanilka/linux-dlc-unlocker-tool/main/ldu-tool.py) в удобное вам место.
2. Установите библиотеку **requests**, если по какой-то причине в вашем дистрибутиве её нет.
```bash
pip3 install requests
```
3. Поместите скрипт в папку с игрой.
4. Дайте скрипту права на исполнение.
```bash
chmod +x ldu-tool.py
```
5. Запустите скрипт.
```bash
./ldu-tool.py install
```

## ⚙️  Список команд
|Команда|Назначение  |
|--|--|
|`./ldu-tool.py -h/--help` `./ldu-tool.py install -h/--help`|Отобразить подсказку по командам|
| `./ldu-tool.py install` | Запускает базовую установку в директории где находится скрипт. Скачивает dlc и модифицированную библиотеку `libsteam_api.so` |
|`./ldu-tool.py install --path/-p <PATH_TO_GAME>`|Запускает базовую установку по указанному пути. Скачивает dlc и модифицированную библиотеку `libsteam_api.so`|
|`./ldu-tool.py install --dlc`|Скачивает **только DLC** с удаленного сервера|
|`./ldu-tool.py install --libs/-l`|Скачивает **только `libsteam_api.so`** с удаленного сервера|

## 🔍 Как это работает?

1.  **Проверка директории:** Скрипт ищет бинарный файл `stellaris` в указанной папке. Если его нет, запрашивает подтверждение у пользователя.
2.  **Эмулятор API:** Загружает свежую сборку `libsteam_api.so` из GitLab-репозитория Goldberg Emulator и помещает её в корень игры.
3.  **Настройка DLC:** Создаёт папку `steam_settings/` и файл `dlc.txt`, заполняя его ID всех официальных DLC через публичный API `api.steamcmd.net`.
4.  **Загрузка контента:** Скачивает ZIP-архивы с файлами DLC с внешнего файлового сервера, указанного в `data.json` в [**данном репозитории**](https://github.com/seuyh/stellaris-dlc-unlocker).
5.  **Распаковка:** Извлекает содержимое архивов в папку `dlc/` и автоматически удаляет исходные `.zip` файлы.
6.  **Запуск (опционально):** После установки предлагает запустить игру через Steam, используя протокол `steam://run/281990`.

## 📂 Структура файлов после установки
```
/путь/к/Stellaris/
├── stellaris              # Основной исполняемый файл игры
├── libsteam_api.so        # Эмулятор Steam API (Goldberg)
├── steam_settings/
│   └── dlc.txt            # Список ID всех разблокированных DLC
└── dlc/                   # Распакованные файлы контента DLC
```

## ❓ Устранение неполадок

-   **Ошибка подключения к `api.steamcmd.net`:** Попробуйте использовать VPN или прокси, если находитесь в регионе с ограничениями доступа.
-   **`PermissionError` при создании папок:** Убедитесь, что у вашего пользователя есть права на запись в папку с игрой. Не запускайте скрипт через `sudo`, лучше измените владельца папки через `chown -R $USER:$USER /путь/к/Stellaris` (рекурсивно меняет владельца папки и всех её вложенных файлов на текущего пользователя).
-   **Игра не видит DLC:** Проверьте, что `libsteam_api.so` лежит в той же директории, что и бинарник `stellaris`, а папка `dlc/` содержит распакованные файлы, а не архивы.
